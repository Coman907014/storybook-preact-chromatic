<p className="row start-align col-lg-12 col-md-12 col-sm-12">The baseline grid is used to create vertical rhythm in digital product design. All components align to 4px square baseline grid for mobile, tablet, and desktop.</p>

<p className="row start-align col-lg-12 col-md-12 col-sm-12">In addition the Virgin Atlantic type stack specifies 16px for the base font size, which produces an 8px x-height.<br/>
The x-height is halved (when needed) to produce a 4px baseline. The 4px baseline is there to allow more flexibility for line heights and smaller adjustments.
All text, components, imagery (everything) flows vertically along this baseline, creating a similar rhythm across all screens in our experience. This rhythm is created by the line height between each line of text and the margin between elements. </p>

<h5 className="row start-align col-lg-12 col-md-12 col-sm-12">Columns</h5>
<p className="row start-align col-lg-12 col-md-12 col-sm-12">Columns define the space inwhich content can be placed on-screen. In responsive layouts, columns are fluid which means that their width is determined but the gutters and margin values. If these values change at each breakpoint, the column width adjusts based on this.</p>
<h5 className="row start-align col-lg-12 col-md-12 col-sm-12">Gutters</h5>
<p className="row start-align col-lg-12 col-md-12 col-sm-12">A gutter is the space between columns, helping to separate content on a page. They are fixed values that can change at each breakpoint to better adapt to a given screen resolution.</p>
<h5 className="row start-align col-lg-12 col-md-12 col-sm-12">Margins</h5>
<p className="row start-align col-lg-12 col-md-12 col-sm-12">Margins are the space either side of the columns on the left and right edges of the screen. They can be fixed or scaling values that can change at each breakpoint to better adapt to a given screen resolution.</p>
<h3 className="row start-align col-lg-12 col-md-12 col-sm-12">Breakpoint System</h3>
<p className="row start-align col-lg-12 col-md-12 col-sm-12">Users expect any website to be perfectly complementary with every single device they own – desktop, tablet, or mobile. If a website’s responsive design does not align with a certain device resolution (especially a commonly used device), the site is at risk of missing out on a segment of its target audience. Therefore, breakpoints help to maintain layout integrity across screen sizes.</p>
